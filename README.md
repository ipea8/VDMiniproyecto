# VDMiniproyecto
## Grupo trabajo Miniproyecto UV


Idea a realizar: paradas de autobuses cerca de localizaciones turísticas de importancia

Las localizaciones de importancia se componen de Recursos turísticos, Monumentos, Zonas de actividad, zonas de juego infantil, fuentes de agua pública, parques/zona de descanso y hoteles.

Esto se relaciona con los datos de las paradas de autobús de la EMT Valencia.

DATASETS A UTILIZAR: Recursos turístics, Monuments, Zones d'activitat, Zones de jocs infantils, fonts d'aigua pública, Cartografia Base Vegetacio, Hotels i estacions d'autobusos EMT. 


Premisa del trabajo: Analizar las zonas de mayor interés para los turistas y la conectividad que tienen con las paradas de autobuses de la EMT
